node-web-cluster
================

Node web cluster boilerplate.

### Getting Started

The project can be fully comtrolled using the NPM scripts defined in package.json.

* Start cluster
```sh
npm start
```

* Debug cluster
```sh
npm run-script debug
```

* Start single server
```sh
npm run-script start-server
```

* Debug single server
```sh
npm run-script debug-server
```
