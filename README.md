node-web-cluster
================

Node web cluster boilerplate, used to create scalable web apps, with improved uptime. 

### Getting Started

The project can be fully controlled using the NPM scripts defined in package.json.

* Start cluster
```sh
npm start
```

* Start single server
```sh
npm run-script start-server
